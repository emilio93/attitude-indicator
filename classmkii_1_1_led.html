Blink led i_u32BlinkCount times for a timer count i_u32TimerCount. This will cause a timer to count up to i_u32TimerCount for 2*i_u32BlinkCount times, toggling the led's output each time. The blinking event will be launched upon calling this function.

Parameters
i_u32BlinkCountQuantity of times the led blinks.
i_u32TimerCountTimer Count.
i_pTimerthe timer to use.
