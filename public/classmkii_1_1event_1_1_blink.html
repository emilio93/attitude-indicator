void mkii::event::Blink::Handler ( void  )
finaloverridevirtual

The blink interrupt handler. This counts down to 0 for each pair of toggles the m_u32BlinkCount, toggling every m_u32TimerCount cicles counts for the used timer. On the last call, the interrupt is unregistered and disabled, the m_bIsBlinking variable is set to false, enabling a new blinking to start.

Implements mkii::IEvent.
